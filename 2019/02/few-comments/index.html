只言片语的评论

Twitter用多了思维总是很碎片化，有些话题想写blog吧，结果想想一两句话就解决了。然后忙着忙着就忘了。但最近未免火热的热点事件也太多了点，想想干脆开个新blog文章，只言片语一两句话，记录一下目前的我目前的想法与评论。

Epic独占 地铁：离乡

谁都记得Xbox独占古墓丽影·崛起那个天大的笑话，说真的发行商抢占平台靠的不是砸钱抢独占，而是靠靠谱的增值服务与合作量。就跟爱奇艺乐视独播的新番一样，没有增值服务没有粘性用户你拿什么保证留存度。可惜了《地铁·离乡》这一作优秀的素质，没有正常价格的主机版或者Steam版之前估计不会玩了。

咪蒙与旗下公众号因为输出焦虑而被封杀

流量时代5000万价值的人血馒头有多少公众号等着去抢，和刚刚从奄奄一息复活的暴走大事件一样，咪蒙又是一个文字狱时代，平台自我审查的牺牲品，当代封闭互联网生态圈的文字狱。我讨厌咪蒙输出的价值观与社会焦虑，但我更讨厌这种自我审查下的太监做派。

公司强迫员工签连带责任书，员工不签没教养

@Makittsp：好容易有个还可以的姑娘，培训3天昨天正式上了一天班，请她签一个大家都签的，如果工作中有失误或错误对公司造成损失担保人有连带责任，的保证书，人家回答她的家人不能写出来，不能承担责任。然后就不来了。
广东籍。
-招个有正常教养的正常人怎么这么难
–同时又一直在叫找工作难

公司是法盲，没有一个懂《劳动法》的而已。员工不是合伙人不是投资人，不需要对工作职责内发生的公司损失承担连带责任。需要承担责任的是任命用人的公司本体而已。没有这基本的保护谁还给公司打工，自己当老板得了，反正责任都是一样的无限大。

赤烛新作《还愿》彩蛋影射政治讽刺，Steam瞬间差评如潮

游戏，特别是独立游戏是很纯粹的东西。无论谁用来夹带私货，都是不可饶恕的恶劣行为。这不是一个政治问题，这是职业道德问题。无论是主创恶意还是美术夹带私货，赤烛大概率凉了，连带着一票独立游戏与刚刚有点起色的中国游戏产业。呜呼悲哉。
更可气的是各路民逗社评还是小粉红云玩家，各路冷嘲热讽夹杂自己的政治观点，非要把这种事情当成自己的枪来扫射一遍才开心么？更可笑的是拿各种言论自由案例，各种影射暗喻来做比较，喂，这事儿低级的仿佛村头老婆婆写个名字塞鞋底踩小人一样，可笑幼稚之极，付出代价理所应当。

暂时想说的也就这么多，想到啥再加。
Fin.

香港——周日，数十万人在香港举行抗议活动，反对政府允许将犯罪嫌疑人引渡到中国大陆的计划。这是香港历史上规模最大的示威活动之一，也是人们对公民自由受到侵蚀的担忧和愤怒上升的又一迹象。

屁股决定头脑，我无法掩饰在这一事件中作为一名大陆人所感受到的被冒犯。其实这表达了香港民众对大陆法律的蔑视与不信任，而且这一点也无法辩驳。但更体现旧世界高人一等可以无视法律的意识形态残留，可以说是对大陆崛起威胁论与不信任最讽刺的案例。
细思恐极的是，如果从上街抗议的那一刻起，暴力抗议与暴力执法就是无可避免的？
香港的没落是有目共睹的，深圳与上海替代了他太多的功能，香港逐步沦为一个中介而已。而且中介的功能还在快速消逝。
